# BizxFlow

An Employee Management System built with Node.js, Express, and MongoDB.

## Features

- Role-based access control (Admin, Manager, Employee)
- Attendance tracking
- Task management
- Leave management

## Tech Stack

- Node.js
- Express
- MongoDB + Mongoose
- JWT Authentication
- Cloudinary (Image uploads)
- Multer

## Setup

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with your environment variables
4. Run `npm run dev`
